## Goal
Ship a runnable React + Vite Cockpit under `frontend/` that renders the hero frame and the four demo-critical decisions from mock data only. Mock data mirrors the backend contract (`CockpitState`) so Milestone 3 swaps `fetch` URLs without redesign.

## Non-negotiables (from `docs/cockpit_visual_direction.md`)
- Editorial / Cartographic aesthetic. Paper background (`#FBF8F2`), ink text, sienna/olive/ochre/gold-leaf accents under 8% of screen.
- Fraunces (display), Instrument Serif (italic accents), IBM Plex Sans (body), IBM Plex Mono (data).
- No Inter, no Roboto, no purple gradients, no bouncy motion, no dark mode.
- Confidence always two decimals, tabular-nums, color-banded.
- Chip radii ‚â§ 2, card radii ‚â§ 8. `atmosphere` shadow only on hero frame canonical card.
